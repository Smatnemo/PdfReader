# PdfReader
This is a document Reader that is built using PyQt5. It is meant to be used as an app on debian distros.

## Platforms
- Ubuntu
- Raspberry Pi

## License
The License is GPLv3

## Installation
    python -m venv venv
    source venv/bin/activate
    python -m pip install -r requirements.txt

## Launching App
    python main.py
